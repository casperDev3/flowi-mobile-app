## Фінанси

- 🟢 **1.1.2** — Заощадження-% «зникає» якщо доходу немає. Показувати «—%» сірим, щоб картка не «дихала».
- 🟢 **1.1.3** — Рядок «Інша валюта» лише з carryover виглядає порожньо. Додати компактну суму поряд.
- 🟢 **1.1.5** — Попап «Всі валюти» без кнопки «Закрити» внизу для нижнього досягання великого пальця.
- 🟢 **1.2.3** — Тікер кастомної валюти приймає будь-які `[A-Z0-9]{≤8}`. Строгий regex + inline-помилка на дублі.
- 🟢 **1.2.4** — При перемиканні income/expense категорія завжди скидається, навіть якщо існує в обох списках.
- 🟢 **1.2.5** — Disabled-стан кнопки «Додати» слабкий візуально. Прозорість 0.4.
- 🟢 **1.3.1** — Поле в «Розподіл балансу» приймає `−5000`. Додати тонку підказку «Можна −».
- 🟡 **1.3.2** — На вузьких екранах три кнопки «Скинути / Скасувати / Зберегти» стискаються. Винести «Скинути» вище.
- 🟢 **1.3.4** — Дубль тікера ловиться лише при `Add` без feedback. Inline-error.
- 🟢 **1.4.2** — Денний підсумок з ≥3 валютами переноситься на новий рядок. Скоротити формат.
- 🟢 **6.x** — Дрібні неконсистентності borderRadius (12 vs 14, 18 vs 20).
